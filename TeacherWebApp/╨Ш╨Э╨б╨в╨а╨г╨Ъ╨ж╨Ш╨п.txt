🚀 БЫСТРЫЙ СТАРТ (3 ШАГА)
═══════════════════════════════════════════════════════════════════════════════

ШАГ 1: Установите Python 3.7 или выше
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Если Python не установлен:
  Windows:  https://www.python.org/downloads/
  Mac:      brew install python3
  Linux:    sudo apt-get install python3 python3-pip

Проверьте установку:
  python --version
  pip --version

ШАГ 2: Установите зависимости
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Откройте командную строку (CMD, PowerShell или Terminal) в папке приложения:

  pip install -r requirements.txt

Это установит:
  - Flask (веб-фреймворк)
  - Flask-CORS (для кросс-доменных запросов)
  - Werkzeug (утилиты для Flask)

ШАГ 3: Запустите приложение
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

В той же командной строке:

  python app.py

Вы увидите:
  ============================================================
  Веб-приложение запущено!
  Откройте в браузере: http://localhost:5000
  ============================================================

Откройте браузер и перейдите на: http://localhost:5000

🐛 РЕШЕНИЕ ПРОБЛЕМ
═══════════════════════════════════════════════════════════════════════════════

ПРОБЛЕМА: "ModuleNotFoundError: No module named 'flask'"
РЕШЕНИЕ:  pip install -r requirements.txt

ПРОБЛЕМА: "Address already in use" (порт 5000 занят)
РЕШЕНИЕ:  Закройте другое приложение на порту 5000 или отредактируйте app.py:
          Найдите строку: app.run(debug=True, host='0.0.0.0', port=5000)
          Измените на:    app.run(debug=True, host='0.0.0.0', port=5001)

ПРОБЛЕМА: "Permission denied" при запуске
РЕШЕНИЕ:  Убедитесь, что файл app.py имеет права на чтение/запись

ПРОБЛЕМА: База данных повреждена
РЕШЕНИЕ:  Удалите файл teachers.db и перезапустите приложение

ПРОБЛЕМА: Приложение не открывается в браузере
РЕШЕНИЕ:  Убедитесь, что в консоли нет ошибок
          Попробуйте http://127.0.0.1:5000 вместо http://localhost:5000

API Endpoints (для разработчиков):

  GET    /api/teachers/<id>            - Информация о преподавателе
  POST   /api/teachers                 - Добавить преподавателя
  PUT    /api/teachers/<id>            - Обновить преподавателя
  DELETE /api/teachers/<id>            - Удалить преподавателя
  GET    /api/statistics               - Статистика

В: Нужна ли база данных MySQL?
О: Нет, используется встроенная SQLite

В: Можно ли использовать на нескольких компьютерах?
О: Да, если они в одной сети. Измените host с '0.0.0.0' на IP адрес

В: Как сделать резервную копию данных?
О: Скопируйте файл teachers.db в безопасное место

В: Можно ли добавить новые кафедры?
О: Да, через API или отредактировав базу данных напрямую

В: Как остановить приложение?
О: Нажмите Ctrl+C в командной строке
